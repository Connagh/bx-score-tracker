# BX Arena

Beyblade X score tracker. Players are nodes in a graph. Drag one onto another to battle.

Battle history, Battle Pass launch speeds over Bluetooth, everything stored locally in your browser.

Tournaments too: tap the trophy, pick bladers and a format (round robin, knockout, or double elimination), and the next scheduled pair lights up in the arena. Tournament battles count toward rivalry records like any other.

One `index.html`. Deploy with GitHub Pages: **Settings → Pages → main / root**.
